# Sakshi Website

Static website for Sakshi Aggarwal, Business Broker working with Clyth McLeod.

## Structure

- `index.html` - home page
- `about.html` - about page
- `readiness-check.html` - interactive readiness tool
- `contact.html` - contact page
- `thanks.html` - thank-you page
- `assets/` - shared CSS and JS

## Deployment

This site is designed for static hosting.

Recommended host: Netlify

The contact form is set up for Netlify Forms.
