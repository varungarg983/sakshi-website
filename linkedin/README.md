# LinkedIn Automation Starter

This folder is a practical LinkedIn workflow for Sakshi Aggarwal.

It is designed to automate:

- topic planning
- post batching
- content structure
- weekly scheduling workflow
- lead-oriented calls to action

It does **not** assume direct API posting from a personal LinkedIn profile.

## Why

LinkedIn automation is easiest and safest when you automate the content system first, then use an approved scheduler or manual posting step.

That gives you:

- consistency
- faster content production
- lower risk than brittle scraping tools
- better quality control

## Folder contents

- `content-pillars.md` - the main LinkedIn content strategy
- `post-templates.md` - reusable post structures
- `30-post-ideas.csv` - ready-to-use content ideas
- `weekly-workflow.md` - a repeatable weekly process

## Recommended workflow

1. Pick 3 to 5 post ideas from `30-post-ideas.csv`
2. Write them using `post-templates.md`
3. Edit them to sound like Sakshi
4. Schedule them in a LinkedIn-compatible scheduler or post manually
5. Track which topics bring profile visits, enquiries, and conversations

## Best starting point

Post 3 times a week:

- one seller post
- one buyer post
- one trust-building personal/professional post

## Important

Keep the goal simple:

- build trust
- generate seller conversations
- generate buyer enquiries
- point people toward the website and Clyth McLeod profile where relevant
